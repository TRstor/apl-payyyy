# 💳 APL Pay - نظام إرسال روابط الدفع

نظام بسيط لإنشاء وإرسال روابط دفع للعملاء عبر البريد الإلكتروني.

## ✨ المميزات

- إرسال رابط دفع مخصص على إيميل العميل
- صفحة دفع احترافية يراها العميل
- عرض اسم المنتج والسعر واسم العميل
- تتبع حالة الدفع (بانتظار / مدفوع)
- تصميم عصري يدعم العربية

## 🚀 التشغيل

### 1. تثبيت المتطلبات

```bash
npm install
```

### 2. إعداد SMTP (البريد الإلكتروني)

انسخ ملف الإعدادات وعدّله:

```bash
cp .env.example .env
```

عدّل ملف `.env` بإعدادات بريدك الإلكتروني:

```env
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=your-email@gmail.com
SMTP_PASS=your-app-password
BASE_URL=http://localhost:3000
```

> **ملاحظة Gmail:** تحتاج إنشاء [App Password](https://myaccount.google.com/apppasswords) من إعدادات حسابك.

### 3. تشغيل الخادم

```bash
npm start
```

افتح المتصفح على: `http://localhost:3000`

## 📖 طريقة الاستخدام

1. افتح لوحة التحكم `http://localhost:3000`
2. أدخل **اسم العميل** و **إيميله** و **اسم المنتج** و **السعر**
3. اضغط **"إرسال رابط الدفع"**
4. يصل العميل إيميل فيه رابط دفع
5. العميل يضغط الرابط ويتم الدفع
